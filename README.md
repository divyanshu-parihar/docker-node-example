
# Docker Express App

This is an Docker Express App containerized.



## Installation

Install my-project with npm

```bash
  npm install my-project
  cd my-project


  // To build container
  docker build . -t <name>
  
  // check for you image
  docker ps

  // run you container
  docker run -p 49160:8080 -d <name>

  Your App should spin up in port 49160.
```
